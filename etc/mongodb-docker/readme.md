# Execució amb docker compose

Per crear l'estructura de contenidors i executar-lo
```
docker-compose up -d 
```

Per aturar l'execució
```
docker-compose stop
```

Por tornar-la a arrencar
```
docker-compose start
```

Per eliminar-lo
```
docker-compose down
```
